# 🎵 GitHub MP3 Player

A modern, browser-based MP3 player that loads and plays music directly from your GitHub repository. Perfect for hosting your personal music collection on GitHub Pages!

## ✨ Features

- **🎶 Play MP3 files** directly from your GitHub repository
- **📱 Responsive design** - works on desktop, tablet, and mobile
- **🔊 Full playback controls** - play, pause, next, previous, shuffle, repeat
- **📊 Progress bar** with seek functionality
- **🎨 Beautiful UI** with glassmorphism effects
- **📋 Automatic playlist** generation from your music folder
- **🔄 Real-time updates** - refresh to see new songs
- **🔒 Privacy-focused** - all processing happens in your browser
